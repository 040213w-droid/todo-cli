# 命令行待办事项工具 (Todo CLI)

## 项目简介

一个基于 Python 的命令行待办事项管理工具，支持任务的增删改查，数据持久化到 SQLite，操作日志自动记录。

**适用场景**：学习 Python 基础、SQLite 操作、项目模块化设计。


## 功能

- 添加待办事项
- 查看所有任务（显示完成状态和创建时间）
- 将任务标记为已完成
- 修改任务内容
- 删除任务（带二次确认，防止误删）
- 操作日志自动记录到本地文件

## 项目结构
```
todo-cli/
├── main.py # 程序入口，菜单展示与用户交互
├── crud.py # 数据库增删改查业务逻辑
├── db.py # 数据库连接管理与初始化
├── text.db # SQLite 数据库文件（自动生成）
├── data.log # 日志文件（自动生成）
└── README.md # 项目说明文档
```


## 快速开始

### 环境要求
- Python 3.7 及以上版本
- 无需安装第三方库（全部使用 Python 标准库）


### 运行步骤

1. 克隆项目到本地
   ```bash
   git clone [你的仓库地址]
   cd [你的项目文件夹名]
    ```
2. 运行程序
    ```bash
    python main.py
    ```

## 设计亮点

1. **模块化分层**：db（数据库连接）、crud（业务逻辑）、main（用户交互）三层分离，职责清晰
2. **资源安全**：使用 `with` 语句管理数据库连接，确保每次操作后自动释放资源
3. **输入校验**：在程序入口统一校验用户输入，防止非法数据进入业务层
4. **日志追踪**：所有操作均有日志记录，方便排查问题
5. **防误删设计**：删除前展示任务内容并二次确认，提升用户体验
